# Roblox BanApi Bypass

## Only works on windows

## Requirements

Python

All python imports are native so no need for pip install bs.

## How to use

Run the python file and select the only option. To make sure it works load the roblox app (not web), if you are logged out then its worked. Simply login and ur good to go. Do this each time you switch account.

## The future

I have a version with account saving so that it's easy to switch accounts without the need to login each time. This will have a cloud system or the option to use local saves.

## How it works

It is very simple, it just clears the file with your roblox cookies. The way the ban api works is that if one account is banned then it will ban all the cookies in that folder. So by clearing all the cookies in that file, it doesn't see a banned cookie.
